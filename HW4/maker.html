  1. To add questions, double click on the item type or drag it to the labeled drag area (look for the dotted lines!) on the right.

  2. ...Drag questions here!

  3. To edit, simply click in the box and start typing

  4. You can use the buttons to move the item to the top or bottom of the list

  5. You can also drag the items up and down

  6. Delete an item from the quiz by clicking on the trashcan

  7. To change the point value of a question, clicking on the number and type the correct value

  8. More options are available on the left

  9. There are hotkeys for each option below.
    Mac users: Use ctrl + opt + [letter, see help]
    Windows users: Use alt + [letter, see help]

  10. Class Hotkey: L

  11. Random Order Hotkey: O

  12. Quiz Time Hotkey: Q

  13. You can also set the available dates to take the quiz

  14. Start Date Hotkey: R

  15. End Date Hotkey: E

  16. Preview Hotkey: P

  17. Publish Hotkey: B

  18. Save Hotkey: S

  19. Clear Hotkey: C

  20. Remember to save frequently!

  21. Hit clear to start over

  22. When you're all done, click 'Publish' to make the quiz available for students

  23. Congrats!
